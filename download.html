Software License¶

The BASIS software is freely available under a BSD-style open source license that is compatible with the Open Source Definition by The Open Source Initiative and contains no restrictions on use of the software. The full license text is included with the distribution package and available online.

System Requirements¶

Operating System: Linux, Mac OS X, Microsoft Windows

Download¶

The source code of the CMake BASIS package is hosted on GitHub from which all releases and latest development versions can be downloaded. See the Release Notes for a summary of changes in each release.

Clone the Git repository as follows:

git clone https://github.com/schuhschuh/cmake-basis.git

or download either a pre-packaged .tar.gz of the latest BASIS release:

  • Download CMake BASIS v2.1.4 as .zip
  • Download CMake BASIS v2.1.4 as .tar.gz
