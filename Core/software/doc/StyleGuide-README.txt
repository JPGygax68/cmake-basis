
STYLE GUIDE

- Limit the number of columns to approx. 80 characters.
- Use two empty lines in between first level sections, whose headings are
  written with uppercase letters only, e.g.,

    1ST FIRST LEVEL HEADING

    Normal text.


    2ND FIRST LEVEL HEADING

    ...

- Use only one empty lines to separate other sections and paragraphs.

- Capitalize second level headings similar to the title of an article, e.g.,

    This is a Heading Following This Rule
