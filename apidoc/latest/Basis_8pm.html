Detailed Description

BASIS utilities of BASIS package.

Note
The Basis.pm module was automatically created by BASIS from the template file Basis.pm.in which is part of BASIS.

This module imports the modules of the BASIS utilities for Perl and exports these utility functions as part of its own package scope next to the functions defined by this module itself. Hence, it is sufficient to only use this module.
