INTRODUCTION
============

  This is a test project for BASIS. The project was created using the
  basisproject tool of BASIS. As it is a test project for BASIS itself,
  it is included in the source tree of BASIS. When running the test,
  the BASIS test project is build using the built BASIS project itself
  as part of which the test is run. Then, the tests contained and build
  as part of the BASIS test project are executed.
